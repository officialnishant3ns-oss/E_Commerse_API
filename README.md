This project delivers a maintainable and extensible e-commerce backend following a clean, modular architecture. It includes controllers, services, utilities, middleware layers, and validation logic to provide secure and efficient data handling. All endpoints support structured responses, error handling, and token-based authentication.

Full API documentation is available via Postman, providing complete details about endpoints, request/response formats, authentication flow, and sample payloads:
<b>Postman Documentation: https://documenter.getpostman.com/view/49372565/2sB3dSQUKj<b>

Features

User Registration and Authentication (JWT)

Role-based access control (Admin/User)

Product CRUD Operations

Category & Sub-category Management

Cart Management (Add, Update, Remove Items)

Order Creation & Tracking

Cloud Storage Support (Image Uploads)

Centralized Error Handling

Input Validation

MongoDB + Mongoose Models

Production-ready folder structure
